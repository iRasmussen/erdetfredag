# Om erdetfredag

Lille Python-script inspireret af [erdetfredag.dk](https://erdetfredag.dk/), der svarer på det eneste rigtige spørgsmål:

> Er det fredag i dag?

Hvis det er fredag, får du “Ja, det er fredag!”.  
Ellers får du “Nej, det er [ugedag].”.

---

## Formål

Scriptet er tænkt som et helt enkelt eksempel på:

- arbejde med datoer og ugedage i Python  
- et lille program, der kan køres fra kommandolinjen  
- en øvelse, som studerende nemt kan læse, ændre og udvide

Projektet bruges primært i undervisning på Erhvervsakademi København (EK).

---

## Krav

- Python 3 skal være installeret

Du kan teste det ved at køre i en terminal:

    python3 --version

---

## Sådan kører du scriptet

Klon repoet og gå ind i mappen:

    git clone https://github.com/iRasmussen/erdetfredag.git
    cd erdetfredag

Kør derefter scriptet (tilpas filnavnet, hvis du bruger et andet end `erdetfredag.py`):

- På macOS / Linux:

      python3 erdetfredag.py

- På Windows:

      python erdetfredag.py
